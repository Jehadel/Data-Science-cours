# Présentation projet fil rouge parcours Masters Data & IA Ynov 2025 - 2026



# Public concerné : 

Tous les étudiants de master des parours Data Engineer & Expert en IA (M1 & M2)



# Déroulement

Ce projet fil rouge est un projet inspiré d'un projet réel sur lesquels les étudiants vont travailler tout au long de leur parcours de Masters. Le projet sera découpé en sous projets thématiques et travaillé dans les différents modules de formation tout au long des deux années de masters



# Description

Le projet consiste à développer une application data, au format web, concernant la gestion, la visualisation de **données relatives aux aliments et à la nutrition** ainsi le développement d'algorithmes d'IA à partir de ces données

Les fonctionnalités attendues sont les suivantes : 

* Permettre à l'utilisateur de saisir manuellement ou d'extraire des données de recettes et d'ingrédients provenant de différentes sources (site web, photo, document pdf)
* Permettre à l'utilisateur de générer une recette à partir de la photo du plat
* Permettre à l'utilisateur de générer ou recommander une recette à partir d'une liste d'ingrédient dont ils dispose
* Proposer un algorithme d'IA permettant de regrouper sous formes de tag similaires les recettes similaires
* Proposer, pour chaque produit, des informations sur la nutrition permettant à l'utilisateur de mieux choisir les produits à acheter : type d'aliments, énergie contenue, nutiriscore, score d'impact environnemental, ...

## Sous projets

### Gestion de projet

Pour ce sous projet, l'objectif est de mettre en place une méthode de gestion de projet qui pourra être réutilisée par les différentes équipes projet qui vont travailler sur le projet.



### Extraction et consolidation de données

Dans ce sous projet, l'objectif consiste à :

* identifier des sources de données pertinentes (données ouvertes, APIs, sites pouvant être scrappés, ...)
* proposer un schéma d'architecture des sources de données et pipelines d'extraction associées
* mettre en place des briques de code pour extraire et pré-traiter  l'ensemble des données
* organiser les données dans un format pouvant être ingéré dans des bases de données

#### Modules concernés : Gestion d'un projet IA, Outils ETL





###

### Structuration & visualisation de données

Dans ce sous projet, l'objectif consiste à :

* proposer un schéma architecture de bases de données pertinente pour le projet
* structurer et verser les flux de données dans une infrastructure de base de données
* gérer la mise à jour des données
* réaliser des briques de code permettant de visualiser des indicateurs relatifs aux données
* proposer un menu de type dashboard pour cette visualisation

#### Modules concernés : Base de données, Algorithmie, Data Science
